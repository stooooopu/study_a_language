# AWS
AWS = 본체  
연결 선 = SSH  
모니터 = Xshell or putty
```
사용자 --------(모니터)------(연결 선)-- AWS
사용자 ---(X shell or putty)---(SSH)--- AWS
```
- Xshell을 이용해서 AWS에 SSH로 접속을 함
